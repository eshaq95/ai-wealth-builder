# AI Wealth Builder

An AI-powered investment and wealth management platform built with Next.js and FastAPI.

## Features

- 📊 Portfolio Management
- 💰 Budget Tracking
- 📈 Investment Analytics
- 🤖 AI-Powered Recommendations
- 📱 Responsive Design
- 🔐 Secure Authentication

## Tech Stack

### Frontend
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts

### Backend
- FastAPI (Python)
- SQLite
- SQLModel
- JWT Authentication

## Getting Started

### Prerequisites
- Node.js 16+
- Python 3.8+
- npm or yarn
